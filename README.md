Chapter 12

How to use the Management Studio for database design

Exercises

<ol>
<li>Use the Management Studio to create a new database called MyWebDB using the default settings. (If the database already exists, use the Management Studio to delete it and then recreate it.)</li>
<li>Use the Management Studio to create the following tables and relationships.</li>
<li>Define the UserID column in the Users table, the ProductsID column in the Products table, and the DownloadID column in the Downloads table as primary keys and identity columns.</li>
<li>In the Downloads table, set the UserID and ProductID column as the foreign keys.</li>
<li>Define the columns so none of them allow null values.</li>
<li>Use the Management Studio to create indexes for the foreign keys in the Downloads table.</li>
<li>Use the Management Studio to create a unique index on the EmailAddress and ProductName columns and a regular index on the DownloadDate column.</li>
</ol>
